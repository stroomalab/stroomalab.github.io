How to edit (quick steps)
  1. Click a button above — it opens GitHub in a new tab.
  2. If asked, sign in to GitHub.
  3. Edit the file. For photos, upload first and then set the path (example: assets/uploads/jane.jpg).
  4. Scroll down and click Commit changes.
  5. Wait ~1 minute and refresh the website page.
Tip: keep this page unlinked from the main site navigation. It’s meant as a private “admin-like” dashboard.
